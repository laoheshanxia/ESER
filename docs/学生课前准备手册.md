# 学生课前准备手册

> 适用对象：大二能源环境专业同学
> 需要时间：约 30 分钟
> 不用怕：不需要 Linux 基础，不需要装过 ROS2，跟着步骤走就行

---

## 一、课前你必须知道的三件事

1. **课堂上不要求你敲代码**——老师全程投影演示，你看、听、跟思路就好
2. **但你最好课前把环境配好**——课后自己玩的时候用得上
3. **配不好也没关系**——有浏览器就能用的备用方案（GitHub Codespaces）

---

## 二、环境选择：走哪条路？

| 如果你… | 推荐方案 | 难度 |
|---------|---------|------|
| 电脑是 Windows 10/11，硬盘还有 20G+ 空间 | ✅ **方案 A：WSL2 安装**（推荐） | ⭐⭐ |
| 装不上 WSL2，或不想装任何东西 | ✅ **方案 B：GitHub Codespaces**（浏览器就行） | ⭐ |

## 三、方案 A：WSL2 安装（Windows 推荐）

### 3.1 启用 WSL2

按 `Win + R` → 输入 `powershell` → 按 `Ctrl + Shift + Enter`（以管理员身份运行）。

在弹出的黑窗口中逐条输入以下命令（每输一条按一次回车）：

```powershell
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
```

```powershell
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
```

然后**重启电脑**。

### 3.2 安装 Ubuntu 22.04

重启后，再次以管理员身份打开 PowerShell，执行：

```powershell
# 查看可用的 Linux 版本
wsl --list --online

# 如果能看到 Ubuntu-22.04，执行：
wsl --install -d Ubuntu-22.04
```

**⚠️ 如果报错**：`无法从 "raw.githubusercontent.com" 提取列表分发。操作超时`

这是因为国内网络问题。改用下面的方法手动下载：

```powershell
# 从清华镜像站下载（国内最快）
Invoke-WebRequest -Uri "https://mirrors.tuna.tsinghua.edu.cn/ubuntu-cloud-images/wsl/22.04/ubuntu-22.04-server-cloudimg-amd64-wsl.rootfs.tar.gz" -OutFile "$env:USERPROFILE\Downloads\ubuntu-22.04.tar.gz"

# 导入到 WSL
wsl --import Ubuntu-22.04 "$env:LOCALAPPDATA\WSL\Ubuntu-22.04" "$env:USERPROFILE\Downloads\ubuntu-22.04.tar.gz" --version 2
```

安装完成后，在开始菜单找到并打开 **Ubuntu 22.04**（或在 PowerShell 输入 `wsl`）。

第一次打开会让你创建用户名和密码——随便设一个，记下来就行。

### 3.3 安装 ROS2 + OpenCV

在 Ubuntu 终端里输入以下命令（一条一条复制粘贴，每输完一条等它跑完再输下一条）：

```bash
sudo apt update && sudo apt upgrade -y
```

```bash
sudo apt install -y locales && sudo locale-gen en_US en_US.UTF-8 && sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8 && export LANG=en_US.UTF-8
```

```bash
sudo apt install -y software-properties-common && sudo add-apt-repository -y universe
```

```bash
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
```

```bash
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
```

```bash
sudo apt update && sudo apt install -y ros-humble-desktop python3-colcon-common-extensions python3-opencv
```

```bash
echo 'source /opt/ros/humble/setup.bash' >> ~/.bashrc && source ~/.bashrc
```

### 3.4 验证安装

输入以下命令，看到版本号就说明装好了：

```bash
ros2 --help
```
**应该看到**：一大段英文帮助信息（有 topic、node、run 等词）

```bash
python3 -c "import cv2; print(cv2.__version__)"
```
**应该看到**：`4.x.x`（几个数字）

```bash
python3 -c "from cv_bridge import CvBridge; print('OK')"
```
**应该看到**：`OK`

全部通过 → 安装成功 🎉

## 四、方案 B：GitHub Codespaces（零安装）

> 什么也不用装，有浏览器就行。课堂备用方案，也可以作为首选。

### 4.1 准备工作

1. 注册一个 **GitHub 账号**（如果还没有）：https://github.com/signup
2. 把邮箱验证了

### 4.2 启动 Codespaces

1. 打开浏览器，访问课程的 GitHub 仓库（老师会给链接）
2. 点击绿色的 **Code** 按钮
3. 选择 **Create codespace on main**
4. 等 2-3 分钟，浏览器里会出现 VS Code 界面，底部是 Ubuntu 终端

> 环境里已经装好了 ROS2 Humble + OpenCV + CvBridge，**不需要任何额外安装**。

### 4.3 验证

在 VS Code 底部的终端中输入：

```bash
ros2 --help
```

看到帮助信息就说明环境可用。

### 4.4 注意事项

- ⚠️ Codespaces **不能连接 USB 摄像头**。但课堂演示用的是模拟图片，完全不受影响
- ⚠️ Codespaces 每月有免费额度（120 小时），上课用完全够
- ⚠️ 不用的时候记得关掉：VS Code 左下角 → Codespaces → Stop Current Codespace

## 五、方案 C：macOS 安装

如果你用 Mac 且是 Intel 芯片，可以装双系统或虚拟机。如果是 Apple Silicon（M1/M2/M3），推荐用 **VMware Fusion** 或 **UTM** 安装 Ubuntu 22.04 ARM 版。

**最简单的方法：直接用方案 B（Codespaces），省事。**

## 六、安装后：跑一下试试

环境装好后，打开两个 Ubuntu 终端窗口：

**终端 1：**
```bash
ros2 run demo_nodes_cpp talker
```
你会看到每秒打印一行 `Publishing: 'Hello World: 1'`

**终端 2（新开一个窗口）：**
```bash
ros2 run demo_nodes_cpp listener
```
你会看到每秒打印一行 `I heard: [Hello World: 1]`

如果两个终端都在输出 → **你的 ROS2 环境完全 OK！** 课上见 🎉

## 七、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `ros2: command not found` | 没 source | 输入 `source /opt/ros/humble/setup.bash` |
| WSL2 安装报 `0x80370102` | BIOS 虚拟化没开 | 重启进 BIOS → 开启 Intel VT-x 或 AMD-V |
| 下载超时/特别慢 | 国内网络 | 换清华镜像源，或用 Codespaces |
| Ubuntu 终端中文乱码 | 缺少中文字体 | 输入 `sudo apt install -y fonts-noto-cjk` |
| 不知道 Ubuntu 终端在哪 | — | 开始菜单搜 "Ubuntu" 或 PowerShell 输入 `wsl` |
